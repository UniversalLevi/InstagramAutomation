TikTok OAuth – Authorization code

Copy the code below and use it in the curl command to get your access token.

Exchange code for token (PowerShell)

Replace YOUR_CLIENT_SECRET with your Client Secret from the Developer Portal, then run:

curl.exe -X POST "https://open.tiktokapis.com/v2/oauth/token/" `
  -H "Content-Type: application/x-www-form-urlencoded" `
  -d "client_key={{ client_key }}" `
  -d "client_secret=YOUR_CLIENT_SECRET" `
  -d "code={{ code }}" `
  -d "grant_type=authorization_code" `
  -d "redirect_uri={{ redirect_uri }}"
